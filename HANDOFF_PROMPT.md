# Handoff Prompt (универсальный)

Скопируйте блок ниже в **конец текущей вкладки**, когда заканчиваете сессию. Агент обновит `SESSION_SUMMARY.md` и при необходимости `RESUME_PROMPT.md`.

---

```
Мы заканчиваем эту сессию. Подготовь handoff для следующего чата.

Нужно:
1. Что уже сделано
2. Какие файлы созданы/изменены
3. Какие решения приняты
4. Что осталось сделать
5. Блокеры / риски / открытые вопросы
6. Следующий лучший шаг
7. Короткий prompt для новой вкладки

Сделай кратко и по делу.
Оформи как SESSION_SUMMARY.md (полный handoff) и обнови RESUME_PROMPT.md (короткий старт для новой вкладки).
Учти PROJECT.md, CHANGELOG.md, BACKLOG.md.
Не коммить без моей команды.
```

---

## Советы

- Перед handoff: `git status` — что не закоммичено.
- Крупные решения дублируйте в `CHANGELOG.md` (дата + списки).
- Следующий шаг — один конкретный (деплой, фича, баг), не список из 10 пунктов.
